Step 1: Initiating a Research Job

To start a job, send a `POST` request to `/api/v1/research` with your query and file URLs. For security, we only accept pre-signed GET URLs, ensuring you retain full data custody. You can also provide an optional `callbackUrl` for webhooks and `metadata` for your own tracking.

Request Body:

{ "query": "Analyze these documents for evidence of medical malpractice...", "files": [ { "fileName": "Exhibit A - Dr. Smith Notes.pdf", "downloadUrl": "https://client-storage.com/path/to/file1?signature=..." } ], "callbackUrl": "https://api.your-app.com/webhook/depogenius-results", "metadata": { "internalCaseId": "case-12345" } }

Immediate Response:

{ "jobId": "eager-research-job-xyz789", "status": "queued", "statusUrl": "https://api.depogenius.com/api/v1/research/eager-research-job-xyz789" }

Step 2: Processing Your Documents

Our proprietary Clarisearch™ pipeline processes your documents using patent-pending technology that achieves fixed-cost analysis at scale. Files are securely downloaded from your temporary URLs and analyzed by our AI models. The system synthesizes all relevant findings into the structured Evidence Packet. Once processing is complete, source documents are permanently and irrecoverably deleted from our environment.

Polling for Progress (Optional):

You can make a `GET` request to the `statusUrl` to check on a job's progress.

{ "jobId": "eager-research-job-xyz789", "status": "processing", "progress": { "totalFiles": 2, "completedFiles": 1, "currentFile": "Deposition of Jane Doe.txt" } }

Step 3: Receiving Your Results

Once the job is complete, we notify you. We recommend using webhooks for efficiency. If you provided a `callbackUrl`, our server sends a `POST` request to it with the job status and a secure, pre-signed URL to download your JSON Evidence Packet.

Webhook Payload:

{ "jobId": "eager-research-job-xyz789", "status": "completed", "resultsUrl": "https://storage.googleapis.com/depogenius-results/final.json?sig=...", "metadata": { "internalCaseId": "case-12345" } }

Polling for Completion:

If polling, the status response will update once complete.

{ "jobId": "eager-research-job-xyz789", "status": "completed", "resultsUrl": "https://storage.googleapis.com/depogenius-results/final.json?sig=..." }
